# Proyecto Aplicaciones Web – Sistema de Control Escolar DESIT

Este repositorio contiene el proyecto de la materia **Aplicaciones Web** (DESIT),
formado por:

- **Backend**: API REST en Django / Django REST Framework  
- **Frontend**: Aplicación en Angular 16  

El sistema permite gestionar usuarios (administradores, alumnos, maestros) y materias, con restricciones de acceso
según el rol.
